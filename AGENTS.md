## Planning rules
- Always show a plan before editing files
- Edit one file at a time
- Run `npm run lint` after every TypeScript change
- Run `npm run build` before declaring done

## Allowed actions
- Reading source files
- Running lint: `npm run lint`
- Running build: `npm run build`

## Always ask before
- Installing packages
- Editing config files (e.g. vite.config.ts, tsconfig.json, drizzle.config.ts)
- Deleting files

## Testing
Ask me which test framework to use before adding tests.
